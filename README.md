# Real-Time Interactive Chat & Game Toy Project

## ✨ 프로젝트 개요
실시간 채팅 + 실시간 위치 반영 기능을 가진 풀스택 토이 프로젝트

## 🛠 기술 스택
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Real-time: Socket.IO
